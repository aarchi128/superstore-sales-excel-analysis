# Superstore Sales & Customer Analysis (Excel)

## Project Overview
This project analyzes retail transaction data to evaluate sales performance, customer behavior, profitability, and seasonal trends using Microsoft Excel.

The objective was to extract business insights using structured data cleaning, PivotTables, and analytical reasoning.

---

## Dataset Information
- Source: Superstore Dataset (Kaggle)
- Records: 9,994 transactions
- Columns: 21
- Level: Transaction-level retail data

---

## Key Analyses Performed

### Revenue Concentration
- Top 20% customers contribute ~48% of total revenue.
- Indicates moderate revenue concentration.

### Repeat vs One-Time Customers
- Over 99% of total revenue is generated by repeat customers.
- Demonstrates strong customer retention.

### Category Profitability
- Technology: Highest margin (~15.6%)
- Office Supplies: Strong margin (~13.8%)
- Furniture: High revenue but low margin (~3.9%)
- Suggests margin improvement opportunity in Furniture.
